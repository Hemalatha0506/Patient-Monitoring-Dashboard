# Health Dashboard Web App

A responsive healthcare dashboard built using HTML, CSS, and JavaScript. The application displays patient details, diagnosis history, vital health metrics, and diagnostic information in an easy-to-use interface.

## Features

- Patient Profile Display
- Diagnosis History Section
- Health Metrics Cards
- Diagnostic List Table
- Responsive Design
- Dynamic Data Rendering using JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript
